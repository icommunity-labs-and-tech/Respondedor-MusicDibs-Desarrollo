# Privaro — Contexto para Respuestas AI

## Sobre Privaro
Privaro es una infraestructura de AI governance diseñada para proteger datos sensibles en interacciones con LLMs. Ofrece herramientas para empresas que necesitan usar inteligencia artificial cumpliendo con regulaciones de privacidad (GDPR, EU AI Act).

## Servicios principales
- **AI Gateway**: Proxy inteligente que sanitiza datos sensibles antes de enviarlos a LLMs
- **Policy Engine**: Motor de políticas para definir reglas de protección de datos
- **Audit Trail**: Registro inmutable de todas las interacciones con LLMs
- **Compliance Dashboard**: Panel de cumplimiento normativo en tiempo real

## Preguntas frecuentes
- ¿Qué datos protege? → PII, datos financieros, datos de salud, y cualquier dato sensible configurable
- ¿Es compatible con mi LLM? → Compatible con OpenAI, Anthropic, Google, y cualquier API REST
- ¿Cumple con GDPR? → Diseñado desde cero para cumplimiento GDPR y EU AI Act
- ¿Cómo se integra? → SDK, API REST, o proxy transparente
